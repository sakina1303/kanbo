# Loom Video Walkthrough Script

## Duration: 3-5 minutes

---

### Introduction (30 seconds)

"Hi! I'm excited to walk you through the Kanban Task Board I built for this take-home assignment.

This is a fully functional task management application with drag-and-drop functionality, persistent storage, and a clean, modern UI."

---

### Feature Walkthrough (2-3 minutes)

**1. Board Overview (20 seconds)**
"Here you can see the three columns: To Do, In Progress, and Done. Each column shows the task count and has empty state messages when there are no tasks."

**2. Adding a Task (30 seconds)**
- Click the + button on the To Do column
- "Let me create a task. I'll add a title and description..."
- Fill in: "Design homepage" / "Create wireframes and mockups for the new landing page"
- Click Add Task
- "Notice the smooth animation and how the task count updates instantly"

**3. Drag and Drop (40 seconds)**
- Hover over the task to reveal the grip handle
- "The drag handle appears on hover. Let me drag this to In Progress..."
- Drag the task to In Progress column
- "You can see the smooth animation, the overlay effect, and how the columns highlight when I hover over them"
- Create another task and demonstrate dragging between multiple columns

**4. Editing a Task (20 seconds)**
- Hover over a task card
- "The edit and delete buttons appear on hover. Let me edit this task..."
- Click the edit button
- Change the title
- Save
- "Changes are saved instantly and persisted to localStorage"

**5. Search Functionality (20 seconds)**
- "The search bar filters tasks in real-time across all columns"
- Type a search term
- Show filtered results
- Clear search to show all tasks again

**6. Persistence Demo (20 seconds)**
- "All tasks are saved to localStorage, so they persist after page reload"
- Refresh the page
- "As you can see, all our tasks are still here"

**7. Responsive Design (20 seconds)**
- "The design is fully responsive..."
- Show mobile view if possible
- "Columns stack vertically on mobile for optimal viewing"

---

### Technical Highlights (30 seconds)

"Quick technical overview:
- Built with React and TypeScript for type safety
- Tailwind CSS for styling with a custom design system
- dnd-kit for accessible drag-and-drop
- localStorage for persistence
- Clean component architecture with separation of concerns"

---

### Closing (20 seconds)

"That's the Kanban Task Board! It meets all the core requirements:
- CRUD operations
- Drag and drop across columns
- Persistent storage
- Responsive design
- Smooth animations and loading states

Thank you for watching, and I look forward to discussing this project further!"

---

## Recording Tips

1. Close unnecessary tabs/apps for clean recording
2. Use a clean browser profile
3. Move slowly when demonstrating drag-and-drop
4. Zoom in if needed for visibility
5. Keep energy positive but professional
